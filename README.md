ZAH KP - ဂျက်ပေါက် အချီကြီး (Jackpot Predictor)
ZAH KP သည် KP ဗေဒင်နည်းပညာ (Krishnamurti Padhati) နှင့် သုတ္တံ၊ ဝိဘုတ္တိ၊ သတ္တန့် ညွှန်းကိန်းများကို အခြေခံ၍ ကံစမ်းမဲ သို့မဟုတ် ထီပေါက်ကိန်းများကို တွက်ချက်ပေးသည့် Web Application တစ်ခု ဖြစ်ပါသည်။
🌟 Features (အဓိက လုပ်ဆောင်ချက်များ)
 * Ascendant Based Calculation: ဇာတာရှင်၏ လဂ် (Ascendant) ကို အခြေခံ၍ တိကျစွာ တွက်ချက်ခြင်း။
 * Precision Inputs: ဒီဂရီ (Degree) နှင့် မိနစ် (Minute) များအထိ ထည့်သွင်းတွက်ချက်နိုင်သဖြင့် ပိုမိုတိကျသော ရလဒ်များကို ရရှိစေခြင်း။
 * Modern UI: အသုံးပြုရလွယ်ကူပြီး မျက်စိပသာဒဖြစ်စေသော Dark Theme Design။
 * Astrological Logic: KP နည်းကျ နှင့် သတ္တန့်အားသာချက်များကို ပေါင်းစပ်အသုံးပြုထားခြင်း။
🚀 Technologies Used (အသုံးပြုထားသော နည်းပညာများ)
 * Frontend: HTML, CSS, JavaScript (or Framework you used)
 * Logic: KP Astrology Algorithms
 * Design: Custom Modern CSS with Gold Accents

🛠️ How to Use (အသုံးပြုနည်း)
 * ဇာတာရှင်၏ လဂ် (Ascendant) ကို ရွေးချယ်ပါ။ (ဥပမာ - ပြိဿရာသီ)
 * သက်ဆိုင်ရာ ဒီဂရီ (Degree) နှင့် မိနစ် (Minute) ကို ဖြည့်စွက်ပါ။
